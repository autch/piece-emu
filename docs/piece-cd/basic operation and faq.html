■■■■　 ドキュメント　■■■■ 
　操作法、ならびにソフトウェア関係のマニュアルです。
　初めてP/ECEを使う時にご参照ください。
　（ハードウェア関係については、HARDSPECを参照してください）
【P/ECE開発環境のインストール】 
　まず、「P/ECE開発環境」をインストールしてください。
　P/ECE付属のCD-ROMをパソコンにセットすると、インストールが始まります。
（始まらない場合は、CD-ROMの中の「setup.exe」をダブルクリックしてください）
　インストール先を指定可能ですが、途中のフォルダ名にスペースが入る場所へは
　インストール出来ません。
【デバイスドライバのインストール】 
　次に、パソコンとP/ECEを結ぶ、「デバイスドライバ」をインストールします。

　P/ECEとパソコンをUSBケーブルを使って接続します。
　・P/ECE側のUSBコネクタは角型ですので注意してください。
　・（通常状態（非スリープ）もしくは、電池を抜いた状態で接続してください）

　接続すると「新しいハードウェアの追加」ウィザードが表示されます。
　「次へ＞」ボタンをクリックして「使用中のデバイスに最適なドライバを検索する（推奨）」
　にチェックが入っていることを確認し、「次へ＞」ボタンをクリックします。
　「検索場所の指定」にチェックを入れて「参照」ボタンをクリックします。

　Windows98の場合
　　（標準の場所にインストールした場合）
　　 c:\usr\piece\drivers\win98を指定します。
　　 インストールされるとPIECE mobile equipment(W2k)と名前が表示されます。

　Windows2000,Me,XPの場合
　　（標準の場所にインストールした場合）
　　 c:\usr\piece\drivers\winxpを指定します。
　　 インストールされるとPIECE mobile equipmentと名前が表示されます。

　　そこに拡張子が「.inf」のファイルがあるので、選択して「開く」ボタンをクリックします。
　　指示に従って「次へ＞」ボタンをクリックしていきます。
　　最後に、デバイスドライバのアップグレードウィザードの完了画面が出ます。
　　デバイスドライバのインストールが完了しました。
　　これでパソコンがP/ECEを認識できるようになりました。
【システムのインストール】 
　最初、P/ECEにはシステムが入っていません。
　必ずパソコンからシステムをインストールして下さい。
　
　※ご注意※
　システムのインストールに失敗すると、P/ECEが起動しない恐れがあります。
　以下の注意事項を必ず守ってください。
　
　・新品の電池をP/ECEにセットしてください。
　・途中で『絶対に』ケーブルを抜いたり、リセットをかけたりしないでください。
　・USBハブの一部の機種におきましては、電源供給の不安定な機種があります。
　　USBハブを接続していて、P/ECEの認識等に問題がある場合は、
　　パソコンのUSBポートに直接接続してください。

　・スタートメニューの「PIECE」から、「P/ECEコミュニケータ」（WinIsd）を
　　起動します。
　・画面の右側に、「ｼｽﾃﾑ ｱｯﾌﾟﾃﾞｰﾄ」ボタンがあるので、クリックします。
　・自動的にシステムの更新が始まります。
　・完了すると、P/ECEが自動的に再起動します。

　システムをインストールした初期状態では以下のアプリケーションが入っています。
　　USB-GamePad … P/ECEをPC用ゲームパッドとして使うためのソフトです。
　　DateAjust   … 日付と時間を設定できます。
【システムの復旧】 
通常、上記のWinIsdを使って、P/ECEへのシステムインストールを行いますが、
この方法を用いてもインストールが出来なかった場合は、以下の方法でもイン
ストールをする事が可能です。
  1. エクスプローラにて、インストールフォルダの下のbinフォルダに移動します。
    binフォルダの中にwinpku.exeがあるので起動します。
  2. 左側のウィンドゥで、ステータスが通信可能である事を確認して、アップデート
    ボタンを押して、P/ECE上の画面を表示に従います。
  3. 画面が反応しなくても、しばらく（２分程度）はリセットなどを行わないで下さい。 
左側のウィンドゥにステータスの表示が無い場合再度、P/ECEをUSBポートに
接続のやり直しを試みます。その状態で、接続しても P/ECE上の画面に表示
が無い場合は以下の手順を行う必要があります。
  1. P/ECEのSELECTボタンと十字キーの↓方向を押下したまま、背面のリセット
    ボタンを押し    ます。必ず２つのキーを押した状態でリセットして下さい。この時
    P/ECE上の画面には何も表示されません。
  2. ここで、「新しいハードウェアの追加」ウィザードが表示された場合、デバイスド
    ライバのインストール方法の指示に従ってUSBドライバをインストールします。
  3. 「最新の情報に更新ボタン」をクリックします。ここでステータスが通信可能に
    なっていない場合、1.の手順が正しく行われていない可能性がありますので、
    P/ECEを再度接続しなおして、数回試して下さい。
  4. アップデートボタンを押して、P/ECE上の画面を表示に従います。
【トラブルシューティング】

▼電源
Ｑ．P/ECEをセルフパワーのUSBハブに繋いで、ハブのアダプターをコンセントに指している
　　時には、ハブをACアダプター代わりに電源を取れるのでしょか?　

Ａ．はい。セルフパワー・USBハブ単体からの給電で電源を取れます。

Ｑ．連続駆動時間はどれぐらいですか？　

Ａ．だいたい８時間と予測されますが、ソフトによって変わります。再生音の音質を良くしたり、
　　画面描画処理を複雑にすると駆動時間は短縮され、その逆やスタンバイモードを併用す
　　ると駆動時間は長くなります。

Ｑ．充電式の電池でも稼動するのでしょうか？　

Ａ．稼働はするかもしれませんが、動作保証外です。通常の単三電池をお使いください。（ア
　　ルカリ電池推奨）

Ｑ．電池交換時にSRAM内容は保持されますか？　

Ａ．SRAMはプログラムの実行エリアとして捕らえていますので、内容の保持は行いません。
　　データの保存は全てフラッシュメモリで行います。

Ｑ．主電源はどうなっているのでしょうか？　

Ａ．主電源のＯＮ・ＯＦＦはありません。しばらくすればスタンバイモードに切り替わるようになっ
　　ています。補足ですが、ＵＳＢケーブルでパソコンと繋がっている時はＵＳＢ側から電源をと
　　るため、電池は消費しないようになっています。

▼仕様
Ｑ．フラッシュメモリ等のメモリ増設はできないのですか？　

Ａ．基本的には出来ません。物理的な改造を行えば出来ない事はないですが、動作保証外と
　　なります。


【基本操作】 
　P/ECEは、単3電池1本で単体で使用することも、
　パソコンと接続してUSB給電により使用することも可能です。
　（この場合、電池が入っていなくてもかまいません）

　アプリケーションの起動はファイルメニューから行います。
　（パソコンと接続している時には、「P/ECEコミュニケータ」（WinIsd）にて、
　パソコン上の実行ファイルを直接起動することも可能です）

　アプリケーション起動中でも、
　「SELECTボタンを押しながらSTARTボタンを0.5秒間押し続ける」ことにより
　必ずファイルメニューに移行します。

　ファイルメニュー、またはアプリケーションから、
　「STARTボタンを3秒間押し続ける」ことによりシステムメニューへ移行します。

　システムメニューでは、以下の項目が設定できます。
　操作は、パッドの上下で項目を選択し、左右で値を調整します。
　「電源OFF」「アプリに戻る」は、ボタンを押すと即座に実行されます。
　※USB接続中に電源OFFを実行しても、即座に電源が入ります。
　※Bボタンでもアプリケーションに戻ります。

　電源を切った後「STARTボタンを１秒以上押してから離す」と、再び電源が入ります。
　※この操作はVer.1.01と変わっていますので注意してください。
　電源を切った際に動作中のアプリケーションがあった場合は、続きの状態から再開されます。

　電源ＯＦＦ
　音量　　　　：0～9
　コントラスト：0～38
　電池残量表示：ON/OFF
　アプリに戻る

　※ご注意※
　・各種システム項目は電池が切れると初期値に戻ります。
　・戻したくない場合には、USBケーブルでパソコンにつないだ状態で電池を交換してください。
　・電池マークの数字は電池からの供給電圧です。
　・1.50前後～0.80前後が駆動可能で、1.00前後を切ると不安定になりますので、早めの
　　電池交換をお勧めします
　・電池が切れる前に、数十秒以内で新しい電池と交換すると時計の設定は保持されます。

　P/ECEへアプリケーションを登録する時は、USBケーブルを使って
　パソコンから専用ソフト「WinIsd」を使って登録します。
　使用法は「WinIsd」のマニュアルご覧ください。

【フラッシュメモリについて】 
　フラッシュメモリの書き換え可能回数は平均10万回程度が目安です。
　フラッシュメモリの寿命による不具合に関しては、保証の範囲外になります。
