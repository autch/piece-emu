■詳細説明

P00
Di電圧制御		 OUT Si-Diの電圧測定(K67)のための電圧をコントロールします。
0 : 休止
1 : 電圧印加(測定可能)
P07
OSC3クッロク制御		 OUT OSC3のクロックをコントロールします。
0 : 48MHz (12MHz×4)
1 : 24MHz (12MHz×2)
P15
LCD SLCK		 OUT LCDコントローラとのインタフェース－シリアルクロック
SCLK3 として使用します。
P16
LCD SID		 OUT LCDコントローラとのインタフェース－シリアルデータ
SOUT3 として使用します。
P20
LCD CSB		 OUT LCDコントローラとのインタフェース－CSB
SRDY3 をコントロールします。 注）P32と直結されています。
0 : 選択
1 : 非選択
P21
LCD RS		 OUT LCDコントローラとのインタフェース－レジスタ選択
RS をコントロールします。
0 : データ
1 : コマンド
P32
LCD CSB		 IN LCDコントローラとのインタフェース－CSB
SRDY3 として使用します。
P33
LCD RESET		 OUT LCDコントローラとのインタフェース－リセット
0 : リセット
1 : 動作
P22
電源モニタ		 IN 電源の供給情報を得ます。
0 : USB
1 : 電池
P23
オーディオ出力		 OUT PWMによるオーディオの出力です。
16ビットタイマーCH1として使用します。
P26
USB クッロク6MHz		 OUT USBコントローラへクロックを供給します。6MHz
16ビットタイマーCH4として使用します。
P27
赤外線出力		 OUT 赤外線出力LEDのコントロール端子です。
0 : 発光
1 : 消灯
P31
オーディオ・パワー		 OUT オーディオ・アンプのパワーをコントロールします。
0 : 動作
1 : 停止
P34
赤外線入力パワー		 OUT 赤外線受光ＩＣのパワーをコントロールします。
0 : 停止
1 : 動作
P35
電源制御		 OUT 電源の供給元をコントロールします。
1 : 必ず電池から
0 : USBからの供給があればUSBから、無ければ電池から
K50
USB SUSPEND		 IN USBコントローラのサスペンド信号です。
K51
USB INT-N		 IN USBコントローラのINT信号です。
K52
赤外線入力		 IN 赤外線受光データです。
K53
ボタンSELECT		 IN ボタン SELECT
0 : 押されている
1 : 放されている
K54
ボタンSTART		 IN ボタン START
0 : 押されている
1 : 放されている
K60
ボタン右		 IN ボタン 右
0 : 押されている
1 : 放されている
K61
ボタン左		 IN ボタン 左
0 : 押されている
1 : 放されている
K62
ボタン下		 IN ボタン 下
0 : 押されている
1 : 放されている
K63
ボタン上		 IN ボタン 上
0 : 押されている
1 : 放されている
K64
ボタンＢ		 IN ボタン Ｂ
0 : 押されている
1 : 放されている
K65
ボタンＡ		 IN ボタン Ａ
0 : 押されている
1 : 放されている
K66
電池電圧ADC		 アナログIN 電池の電圧が測定できます。
K67
Di電圧ADC		 アナログIN Si-Diの電圧が測定できます。
